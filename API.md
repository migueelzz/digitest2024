### CRUD Products 
[GET] `/products` -> Listar todos os produtos
[GET] `/products/{id}` -> Listar um produto pelo id
[POST] `/products` -> Criar um produto
 - name
 - price

[PUT] `/products` -> Atualizar um produto pelo id
[DELETE] `/products` -> Deletar um produto pelo id
